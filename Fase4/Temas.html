Introducción

Actualmente existen infinidades de etiquetas y elementos HTML para realizar un maquetado optimo y preciso, de esto se hablaremos en este sitio web, orientando y creando bases rígidas a futuros diseñadores de páginas web.

Etiquetas, listas y enlaces 

En la maquetación y diseño de un sitio web es importante tener claro los conceptos de etiquetas, listas y enlaces HTML. A continuación hablaremos de ellos.

Imagen: Nicepage.com

Etiquetas

Las etiquetas semánticas ayudan a definir la estructura del documento y permiten que las páginas web sean mejor indexadas por los buscadores. Una etiqueta se califica como semántica si tiene que ver el significado, es decir, si nos informa sobre lo que trata su contenido.

Listas

Las listas en HTML nos permite crear conjuntos de elementos en forma de lista dentro de una página, todos los cuales irán precedidos, generalmente, por un guión o número.
