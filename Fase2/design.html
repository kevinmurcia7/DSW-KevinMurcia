Proposito del sitio

En esta pequeña página web se dará información sobre el bueno uso y manejo de las etiquetas semánticas de HMTL junto a elementos esenciales como listas y enlaces.

Etiquetas semánticas:

Las etiquetas semánticas ayudan a definir la estructura del documento y permiten que las páginas web sean mejor indexadas por los buscadores.

Listas:

Las listas en HTML nos permite crear conjuntos de elementos en forma de lista dentro de una página, todos los cuales irán precedidos, generalmente, por un guión o número.

Enlaces:

Los enlaces en HTML nos permiten de una manera muy cómoda redirigir al usuario a otra parte, cuando éste pinche en dicho enlace.
